# 论文思路

## 数据准备

*MM4flow: A pre-trained multi-modal model for versatile network traffic analysis*

1. **DataCon2020**：加密恶意软件检测（恶意 vs 正常）。
2. **DataCon2021-p1**：加密代理流量分类（V2ray, Clash, Shadowsocks 等）。
3. **DataCon2021-p2**：加密隧道下的网站识别（通过代理访问网站产生的流量）。这是最难的任务，因为几乎没有明文特征。
4. **Browser**：浏览器分类（Chrome, Firefox, Safari 等）。
5. **NUDT_MobileTraffic**：移动应用识别（300 类 App）。
6. **CSTNET-TLS1.3**：TLS 1.3 协议下的网站识别。
